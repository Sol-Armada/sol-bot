Step 1: Discord Bot Configuration

Configure your Discord bot credentials. You can obtain these from the Discord Developer Portal.

How to get Discord credentials
1. Visit discord.com/developers/applications
2. Create or select your application
3. In OAuth2 → General, copy your Client ID and Client Secret
4. In Bot settings, copy your Bot Token
5. In Discord, right-click your server → Copy Server ID
